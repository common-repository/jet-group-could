== Description ==

en: Create a cloud groups your social network! You have a lot of communities? Want to show users a maximum of groups? Install this widget!
Provides random groups avatar with link to group

ru: Создайте облако групп Вашей социальной сети! У Вас множество групп? Хотите показать пользователям максимум групп? Установить этот виджет!


<a href="http://sportactions.ru">live demo</a> (need registration)

! backward compatibility with 1.1.3

== Installation ==

1. Upload `j-rnd-groups.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add widget to site
